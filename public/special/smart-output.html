Smart Output Feature

What is Smart Output?

Smart Output is a powerful productivity feature in Zoho Notes that streamlines your coding workflow. It decides how results are displayed in your notebook output window.

Smart Output: ON (Balanced Mode)

In this mode, the editor automatically captures and displays the value of the last expression in your code cell, even if you don't use console.log().

Input Code
const a = 10;
const b = 20;
a + b;
Output
30

Smart Output: OFF (Strict Mode)

When turned off, the application acts like a traditional compiler. It will only show output that you explicitly print using console.log(). If nothing is printed, it will remind you.

Input Code
const result = 2 + 2; // Nothing printed
Output
Program did not output anything!

Why use it?

Smart Output is perfect for quick calculations and rapid prototyping where you don't want to type console.log() for every result. Turn it OFF when you want a clean, standard terminal experience where you have full control over what is displayed.
