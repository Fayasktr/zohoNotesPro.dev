Zoho Notes: User Manual

v2.0 Update Docker Enabled

Your ultimate guide to mastering the world's most developer-friendly note-taking platform.

What is Zoho Notes?

Zoho Notes isn't just a text editor; it's a Computational Notebook. It combines the ease of a simple note-taking app with the power of a professional IDE.

Whether you are documenting complex logic using Markdown or testing snippets in JavaScript, Zoho Notes provides a unified, secure environment accessible from any device.

1. Authentication & Access

Cloud Sync & Persistence

The Purpose: To ensure your workspace is available 24/7. When you log in, your notes are pulled from our secure database.

What you can do: Start a note on your computer at home, and continue exactly where you left off on your phone or tablet.

Forgot Password

The Purpose: Security and recovery. If you lose your keys, we provide a 100% safe way to get back in.

What you can do: Request a recovery link to your registered email to reset your credentials securely.

2. File System & Tree View

Zoho Notes features a modern Tree View explorer, allowing infinite nesting and organization.

- Folders: Hover over any folder to access Add File, Rename, or Delete options.
- Nested Structure: Create sub-folders naturally (e.g., naming a folder "Project/Backend" creates a nested tree).
- Visuals: Clear "L-shaped" connector lines help you track your depth in the hierarchy.

3. Dynamic Interaction

Drag & Drop Logic

Inside a File: Grab the vertical handle next to a note to reorder it. Each note has a unique Sequence Number (1, 2, 3...) that updates in real-time as you move things around.

Across Files: Want to move a specific note to a different notebook? Just drag it from the main area and drop it onto a notebook title in the Sidebar.

4. The Safety Net (Trash)

Mistakes happen. Our Trash system acts as a temporary buffer for everything you delete.

Auto-Delete Policy

Items in the trash are kept safe for 15 Days. After that, our automated system permanently purges them to save space.

Restoration

You can restore any notebook or cell. The system remembers where it came from (original path) so it returns exactly where it belongs.

5. The Execution Engine

Monaco Power (VS Code Core)

We use the same core engine that powers VS Code. This means you get:

Auto-Complete
Syntax Highlighting
Multi-Cursor
Bracket Matching

Multi-Language Engine NEW

Zoho Notes is no longer just for JavaScript. We now support full execution for:

  • • JavaScript: Standard V8 Node.js runtime.
  • • Python 3: Full system Python 3 support.
  • • Java (JDK 17): Full compilation & execution. Note: Use 'public class Main' for entry points.
  • • C / C++: Compiled using GCC/G++ with full standard library support.

Docker Isolation

Security First: Every piece of code you run is executed inside a Docker Container. This means the code is completely isolated from the main server, ensuring maximum safety and consistency across different languages.

Smart Output Efficiency

If turned ON, our engine captures the last expression of your code and displays it automatically (JS only). For other languages, use standard print commands like printf or System.out.println.

Batch Operations

- Run All: Executes every cell in the file sequentially.
- Clear All: Wipes all outputs if you want a clean slate.

6. Advanced Safety (Pro Protection)

Rate Limiting & Brute Force Prevention

If three failed logins are detected, the system forces a 30-second cooldown. This makes it impossible for automated attackers to guess your password.

IP Blocking

Our server monitors traffic. If a single IP address sends suspicious amounts of requests, it is automatically throttled or blocked to keep the service fast for everyone.

CSRF & XSS Shield

Every interaction is validated with secure tokens, and every input is sanitized to prevent malicious scripts from affecting your data.
